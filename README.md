# T-Библиотека

Программа для ведения личной библиотеки книг.

Данные сохраняются в обычный текстовый файл "my_library.txt"
(каждая книга — одна строка, поля разделены запятой)

## Возможности

1. Добавить книгу  
2. Просмотреть все книги (с выбором сортировки или фильтра)  
3. Показать избранное  
4. Поменять статус книги (прочитано / не прочитано)  
5. Добавить книгу в избранное  
6. Убрать книгу из избранного  
7. Удалить книгу  
0. Выход

## Как выглядит список книг

Пример вывода:

Колобок, Народное творчество, сказка, -, круглый хлебец убегает от всех, прочитано, да
Теремок, Народное творчество, сказка, -, зверушки живут в теремке, прочитано, да
Репка, Народное творчество, сказка, -, дед тянет репку, не прочитано, нет
Винни-Пух и все-все-все, Алан Александр Милн, сказка, 1926, приключения медвежонка и его друзей, не прочитано, да
Гарри Поттер и философский камень, Джоан Роулинг, фэнтези, 1997, мальчик-волшебник узнает о своем прошлом, прочитано, да
Алиса в Стране чудес, Льюис Кэрролл, фэнтези, 1865, девочка попадает в удивительный мир, прочитано, да
Денискины рассказы, Виктор Драгунский, рассказы, 1960, смешные истории из жизни мальчика Дениски, прочитано, да


Поля всегда идут в таком порядке:  
**название, автор, жанр, год, статус, в избранном (да/нет)**

## Как запустить

1. Установи Python 
2. Сохрани код в файл, например `Task_2.py`
3. Открой командную строку
4. Перейди в папку с файлом
5. Запусти программу:
python Task_2.py

Данные хранятся в файле my_library.txt в той же папке, где запущена программа.
Если файл удалить — библиотека очистится.

Если в названии, авторе, жанре или описании будет запятая, она заменяется на пробел при сохранении (чтобы не сломался формат файла)
Год можно вводить любой текст, но сортировка по году работает только с числами
Поиск и фильтр по жанру — точное совпадение (без учёта регистра)
Номера книг начинаются с 1
